This is a video of the neural network I created in action. It is a 3-layer neural network, taking the (x,y) coordinate of the target as input, and outputting the delay and activation times for each muscle spring in the arm. This allows it to create all motor commands at once instead of fine-tuning as the arm moves closer to the point. While this is not a true model of how limbs are controlled, it simplifies the control algorithm, while still allowing different movements over time.

For training data I created activations for the arm and then recorded where the tip of the arm ended up in the coordinate space. Initially I created random delays and activations to move the arm, but this created too noisy a dataset to train the neural network on. I then randomly generated one or two bend points in the arm, and then created delays and activations to bend the arm at those points to create a more ordered dataset to train the neural network on.
